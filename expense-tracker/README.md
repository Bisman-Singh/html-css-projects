# Expense Tracker

A full-featured expense tracker with charts, built with pure HTML, CSS, and JavaScript.

## Features

- **Add transactions** — income or expense with amount, category, date, and description
- **Summary cards** — balance, total income, and total expenses
- **Pie chart** — spending breakdown by category (Canvas-drawn, zero dependencies)
- **Bar chart** — daily spending for the last 7 active days (Canvas-drawn)
- **Transaction list** — with delete functionality
- **Filters** — filter by category and date range
- **Export to CSV** — download all transactions as a CSV file
- **Persistent storage** — all data saved to localStorage
- **Dark theme** with colorful chart segments

## Usage

Open `index.html` in any modern browser. No build tools or dependencies required.

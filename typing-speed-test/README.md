# Typing Speed Test

A 60-second typing speed test with real-time feedback, stats tracking, and localStorage history.

## Features

- 14 different paragraphs chosen at random
- Real-time character highlighting: green for correct, red for incorrect
- Live stats: WPM, accuracy %, characters typed
- 60-second countdown timer with visual progress bar
- Results screen with WPM comparison to average (40 WPM)
- Past attempts saved in localStorage (up to 20)
- Restart button for a fresh paragraph
- Paste prevention to keep it fair
- Dark theme with monospace font display

## Usage

Open `index.html` in any modern browser and start typing.

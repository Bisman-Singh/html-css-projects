# MadeupAI — Animated Landing Page

A modern SaaS-style landing page for a fictional AI productivity platform, built with pure HTML, CSS, and JavaScript.

## Sections

- **Hero** — Typing effect tagline, floating gradient orbs, stats
- **Features** — 3 cards with icons and hover tilt effect
- **How It Works** — 3-step process cards
- **Testimonials** — 3 customer review cards
- **Pricing** — 3 tiers (Starter, Pro, Enterprise) with highlighted popular tier
- **CTA** — Call-to-action with animated gradient border button
- **Footer** — Brand info and link columns

## Animations & Interactions

- Hero text typing effect (character by character)
- Floating gradient orbs with CSS keyframe animations
- Scroll-triggered fade-in / slide-up using Intersection Observer
- Feature cards: 3D tilt effect on hover (perspective + rotateX/Y)
- Pricing cards: scale-up on hover, popular tier pre-scaled
- Parallax scrolling on hero background orbs
- Animated gradient border on CTA button
- Sticky navbar with blur background on scroll
- Smooth scroll to sections
- Mobile hamburger menu with slide-in panel

## Tech Stack

- HTML5 semantic markup
- CSS3 (custom properties, gradients, animations, grid, flexbox, backdrop-filter)
- Vanilla JavaScript (Intersection Observer, DOM manipulation)

## Usage

Open `index.html` in any modern browser. No build tools, frameworks, or external libraries required.

# Kanban Task Board

A drag-and-drop Kanban board built with pure HTML, CSS, and JavaScript.

## Features

- **Three columns**: To Do, In Progress, Done
- **Add tasks** with title, optional description, and priority level
- **Drag and drop** tasks between columns using the HTML Drag and Drop API
- **Inline editing** — double-click a task title to edit it
- **Delete tasks** with a confirmation dialog
- **Task counts** displayed per column
- **Priority labels** — color-coded: Low (green), Medium (yellow), High (red)
- **Persistent storage** — all data saved to localStorage
- **Dark theme** with smooth transitions

## Usage

Open `index.html` in any modern browser. No build tools or dependencies required.

# Recipe Finder

Search and discover recipes from around the world using TheMealDB free API.

## Features

- Search recipes by name or ingredient
- Filter by category dropdown (loaded from API)
- "Random Recipe" button for discovery
- Recipe cards with image, title, category, and cuisine
- Full recipe detail modal: ingredients, instructions, YouTube link
- Favorites saved to localStorage (heart icon toggle)
- Loading skeleton animation while fetching
- Responsive grid layout
- Dark theme with warm amber/orange accents

## API

Uses [TheMealDB](https://www.themealdb.com/api.php) free API — no API key required.

## Usage

Open `index.html` in any modern browser. An internet connection is required to fetch recipes.

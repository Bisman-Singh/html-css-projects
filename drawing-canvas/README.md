# Drawing Canvas

A full-featured drawing application built with pure HTML, CSS, and JavaScript using the Canvas API.

## Features

- **Drawing Tools**: Pencil (freehand), Line, Rectangle, Circle, Eraser
- **Color Picker**: Native color input + 10 preset color swatches
- **Brush Size**: Adjustable from 1px to 50px
- **Opacity Control**: Adjustable drawing opacity
- **Fill / Stroke Toggle**: Choose between filled or outlined shapes
- **Undo / Redo**: Full history support (up to 50 states)
- **Clear Canvas**: With confirmation dialog
- **Export as PNG**: Download your artwork
- **Grid Overlay**: Optional helper grid
- **Keyboard Shortcuts**:
  - `Ctrl+Z` — Undo
  - `Ctrl+Y` — Redo
  - `Ctrl+S` — Save/Export as PNG
  - `P` — Pencil tool
  - `L` — Line tool
  - `R` — Rectangle tool
  - `C` — Circle tool
  - `E` — Eraser tool
- **Touch Support**: Works on tablets and touch devices
- **Responsive Layout**: Adapts to different screen sizes

## Usage

Open `index.html` in any modern browser. No build tools or dependencies required.

## Tech Stack

- HTML5 Canvas API
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (ES6+)

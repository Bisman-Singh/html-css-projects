# Portfolio Website

A modern, responsive developer portfolio with dark theme, smooth scroll animations, and interactive elements.

## Features

- Responsive CSS Grid layout with mobile hamburger menu
- Scroll-triggered fade-in animations via Intersection Observer
- Animated stat counters and skill progress bars
- Project filtering by category
- Contact form with submission feedback
- Pure HTML/CSS/JS — no frameworks or build tools

## Usage

Open `index.html` in any modern browser.

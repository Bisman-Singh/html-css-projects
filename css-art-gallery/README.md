# CSS Art Gallery

A gallery page showcasing 8 pure CSS illustrations — no images, no SVGs, just HTML and CSS.

## Art Pieces

- **Sunset Scene** — gradient sky with glowing sun and water reflection
- **Coffee Cup** — steaming cup with rising steam animation on hover
- **Rocket Ship** — rocket with flickering flame animation
- **Cat Face** — cute cat with blinking eyes on hover
- **Mountain Landscape** — night mountains with twinkling stars
- **Gamepad Controller** — game controller with glowing buttons on hover
- **Camera** — camera with flash effect on hover
- **Vinyl Record** — record that spins on hover

## Features

- Responsive grid layout
- Hover animations on each art piece
- Lightbox modal for enlarged view
- Dark theme with colorful CSS art
- Pure HTML/CSS/JS — no frameworks or build tools

## Usage

Open `index.html` in any modern browser.

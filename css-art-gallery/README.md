## Features

- **200 unique CSS art pieces** built with gradients, clip-path, box-shadow, pseudo-elements, and keyframe animations
- **Category filter** — browse by any of the 20 categories
- **Live search** — filter pieces by title or category
- **Hover animations** — each piece has idle and hover-activated effects
- **Lightbox modal** — click any piece for an enlarged view
- **Responsive grid** — adapts from mobile to desktop
- **Dark theme** with colorful art on a dark canvas
- **Zero dependencies** — pure HTML, CSS, and vanilla JavaScript

# HTML, CSS & JavaScript Projects

A collection of 13 frontend projects built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no build tools. Open any `index.html` in a browser to run.

## Projects

| # | Project | Description | Key Concepts |
|---|---------|-------------|-------------|
| 1 | [Portfolio Website](portfolio-website/) | Responsive developer portfolio with dark theme | CSS Grid, Intersection Observer, animations |
| 2 | [Weather Dashboard](weather-dashboard/) | Real-time weather with 5-day forecast | Fetch API, async/await, Geolocation |
| 3 | [Markdown Previewer](markdown-previewer/) | Live markdown-to-HTML editor | Custom parser, split-pane layout |
| 4 | [Pomodoro Timer](pomodoro-timer/) | Productivity timer with SVG progress ring | SVG animation, Notification API, AudioContext |
| 5 | [Interactive Quiz App](quiz-app/) | Timed trivia quiz with scoring | State machine, timer, localStorage |
| 6 | [Kanban Task Board](kanban-board/) | Drag-and-drop task management | HTML Drag & Drop API, localStorage |
| 7 | [Expense Tracker](expense-tracker/) | Budget tracker with charts | Canvas charts, CSV export, filtering |
| 8 | [Music Visualizer](music-visualizer/) | Audio visualizer with multiple modes | Web Audio API, Canvas, AnalyserNode |
| 9 | [CSS Art Gallery](css-art-gallery/) | Gallery of pure CSS illustrations | CSS art, clip-path, pseudo-elements |
| 10 | [Typing Speed Test](typing-speed-test/) | 60-second typing challenge | Keyboard events, WPM calculation |
| 11 | [Recipe Finder](recipe-finder/) | Search recipes from TheMealDB API | REST API, favorites, skeleton loading |
| 12 | [Drawing Canvas](drawing-canvas/) | Full-featured drawing application | Canvas API, undo/redo, PNG export |

## How to Run

```bash
# Just open any project's index.html in your browser
open portfolio-website/index.html
```

No dependencies. No build step. No server required.
